# url-relative
calculate RFC 1808 relative URLs (inverse of url.resolve)

## usage
```js
var urlRelative = require('url-relative')
```


## api


## installation

    $ npm install url-relative


## running the tests

From package root:

    $ npm install
    $ npm test
